## “开”消息 - 消息测试工具
一个用于调试消息的插件，可返回消息的详细信息

### 功能特性

- 支持通过引用消息或提供消息ID来获取消息的调试信息
- 解析并展示多种消息类型，包括图片、视频、音乐、语音、@用户、引用消息、位置信息和JSON数据
- 返回消息的字符串表示、消息ID、所有消息类型以及各种媒体链接等详细信息
- 支持地理位置信息的格式化显示
- 提供友好的错误提示

### 安装方法
#### 使用设置向导（插件中心）安装
在设置向导 -> 插件中心页面，切换筛选条件为 ```筛选：全部``` ，然后搜索 ```CheckMessages``` ，点击安装

#### 手动安装
1. 下载 `CheckMessages.py` 文件
2. 将文件放入机器人的 `plugins` 目录
3. 重启机器人或发送 `重载插件` 命令

### 使用方法

在群聊中使用命令：`调试消息【引用一条消息/消息ID】`

例如：`调试消息 123456789` 或引用一条消息后发送 `调试消息`

### 配置说明

- 插件依赖于 Hyper 库
- 需要确保配置文件 config.json 存在且可访问
- 插件会使用配置中的 reminder 字段作为命令前缀

### 注意事项

- 请确保引用的消息是完整的非空消息
- 若消息不存在或为空，会返回错误提示
- 插件仅在群聊中使用，通过群消息事件触发
