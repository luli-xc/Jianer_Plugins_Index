# 群待办插件 (Group Todo Plugin)

一个简单的 QQ 群机器人插件，用于快速设置群待办事项。

## 功能说明

- 在群内回复任意消息并发送"['reminder']群待办"，即可将该消息设置为群待办
- 支持白名单功能，可限制只在特定群聊中使用

## 使用方法

1. **设置群待办**：回复一条消息后，发送 `['reminder']群待办`
2. **查看帮助**：直接发送 `['reminder']群待办` 可查看帮助信息

## 配置说明

编辑 `Grouptodu.py` 文件中的配置项：

```python
# 白名单开关：True=启用白名单，False=禁用白名单（所有群都可以使用）
WHITE_LIST_ENABLED = False

# 白名单，只有在白名单中的群聊才会响应
WHITE_LIST = [
    "114514",
    "350234"
]
```

## 安装方式

1. 将 `Grouptodu.py` 放入 Plugins 插件目录
2. 重载插件即可自动加载

## 注意事项

- ⚠️ **机器人必须拥有群管理员权限才能正常使用**
- ⚠️ **设置群待办需要管理员权限，机器人必须是群管理员**
- 设置群待办时，请确保回复的是需要设为待办的消息

## 错误提示

| 提示信息 | 说明 |
|---------|------|
| ✅ 已将此消息设为群待办啦 | 设置成功 |
| ❌ 设置群待办失败 | 权限不足或其他错误 |
| ⚠️ 请回复一条消息后发送'['reminder']群待办'命令 | 未回复消息就发送了命令 |
