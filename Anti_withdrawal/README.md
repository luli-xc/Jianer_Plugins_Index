## Anti-withdrawal
一个在群聊内管理员撤回群成员消息时发送日志的插件

### 功能特性

- 检测群聊中的撤回事件
- 当管理员撤回群成员消息时，自动发送撤回日志
- 日志包含撤回群信息、撤回者信息、被撤回者信息和撤回时间
- 支持获取用户昵称和群组名称，使日志更加详细

### 安装方法
#### 使用设置向导（插件中心）安装
在设置向导 -> 插件中心页面，切换筛选条件为 ```筛选：全部``` ，然后搜索 ```Anti-withdrawal``` ，点击安装

#### 手动安装
1. 下载 `Anti-withdrawal.py` 文件
2. 将文件放入机器人的 `plugins` 目录
3. 重启机器人或发送 `重载插件` 命令

### 使用方法

插件会自动监听群聊撤回事件，无需手动触发。当检测到非本人撤回的情况时，会向根用户发送撤回通知。

### 配置说明

- 插件依赖于 Hyper 库和 Tools.websocket_message 模块
- 需要确保配置文件 config.json 存在且可访问
- 插件会使用 ROOT_User[0] (即 config.json 中配置的 `owner`) 作为通知接收者

### 注意事项

- 插件需要网络连接以获取用户信息和群组信息
- 仅在管理员撤回群成员消息时触发，用户撤回自己的消息不会触发
- 若获取消息信息失败，会在控制台输出错误信息
